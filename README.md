# Kittygram

Социальная сеть для обмена фотографиями любимых питомцев.

### Автор:
- Yaroslav Baramykov

### Технологии:
- Python 3.9
- Бэкенд-приложения на Django
- Фронтенд-приложения на React

###

Социальная сеть позволяет:
* Опубликавать фото котика
* Указать имя и возраст
* Поделиться достижениями котика

### Как попасть на сайт:
